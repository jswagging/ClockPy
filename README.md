## How do I make themes?
To make themes, they must follow this formula:

        "TemplateFormula": """
        QWidget { background:#ffffff; color:#ffffff; }
        QTabBar::tab { background:#ffffff; }
        QTabBar::tab:selected { background:#ffffff; color:#ffffff; }
        QPushButton, QLineEdit { background:#ffffff; }
    """,

  After making your theme, send it to jamercontactemail@gmail.com. I will respond with wether if it's accepted or not.
